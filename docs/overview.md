# システム概要

- 指定されたAPIでアクセスできるドキュメントからRAG（Retrieval-Augmented Generation）サーバーを構築し、LINE Botを通じてユーザーと対話する
- "sync" とメッセージを送信すると、設定されたAPIからすべてドキュメントを取得し、インデックスを更新する
- コマンド以外のメッセージを送信すると、RAGから回答を取得して返信する
- ドキュメントの一覧を取得するAPIのエンドポイントとURLパラメーター、ページネーション方法（オフセット、カーソル等）を設定可能
- ドキュメントの内容を取得するAPIのエンドポイントとURLパラメーターを設定可能
- ドキュメントの内容の形式を設定可能
    - JSON: タイトルや本文のフィールド名を指定可能（階層構造も対応）
    - HTML: タイトルや本文のセレクタを指定可能
    - プレーンテキスト
    - その他必要に応じて拡張可能
- 各種設定は一旦環境変数から読み込み、DIコンテナに渡す
